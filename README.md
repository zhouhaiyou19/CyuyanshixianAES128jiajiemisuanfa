本仓库提供了一个使用C语言实现的AES128加解密算法资源文件。该项目详细讲解了AES128加解密的实现过程，并提供了完整的项目结构、使用环境、编译方法以及功能说明。此外，还附带了辅助资料和项目流程图，帮助用户更好地理解和使用该算法。

## 项目内容

- **README文件**：详细介绍了项目的总体结构、使用环境、编译方法和功能。
- **辅助资料**：提供了与AES128加解密相关的额外资料，帮助用户深入理解算法原理。
